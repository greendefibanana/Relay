# Glossary

## AssetRegistry

BOLT component that tracks public or settlement-facing ownership state.

## BOLT ECS

Entity-component-system framework used by Relay to separate protocol state and execution logic.

## BuyerClearance

Component that represents buyer eligibility before a match can execute.

## DealTerms

Confidential component that stores private listing and RFQ constraints.

## MagicBlock PER

Private Ephemeral Rollup execution environment used by Relay for private state execution.

## RFQ

Request for quote. A negotiation model where buyers submit bids against seller-defined constraints.

## SAFT

Simple Agreement for Future Tokens.

## SAFE

Simple Agreement for Future Equity.

## TEE

Trusted Execution Environment. Hardware-backed environment used for private execution.

## Vested Token

Token allocation that unlocks over time according to a vesting schedule.

## Whale-to-Whale OTC

Private negotiation between large holders or buyers where public order flow would be undesirable.
